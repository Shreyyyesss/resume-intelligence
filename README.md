\# Resume Intelligence



Resume Intelligence is an AI-powered resume analysis tool that helps users evaluate and improve their resumes using modern NLP and AI techniques.



\## Features

\- Resume Upload (PDF/DOCX)

\- Resume Parsing

\- Skill Extraction

\- Job–Resume Matching

\- ATS Score

\- AI Resume Feedback

\- Keyword Suggestions



\## Tech Stack

\- Next.js

\- React

\- TypeScript

\- TailwindCSS

\- NLP Processing



\## Installation



npm install

npm run dev
